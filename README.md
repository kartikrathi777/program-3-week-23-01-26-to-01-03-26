# program-3-week-23-01-26-to-01-03-26
Given a string s consisting of balanced parentheses, calculate the score of the string based on the following rules:  "()" has a score of 1. "AB" has a score of A + B, where A and B are balanced parentheses strings. "(A)" has a score of 2 × score(A), where A is a balanced parentheses string.
